# DeepSeek-VectifyAI-PageIndex

## 项目介绍
DeepSeek-VectifyAI-PageIndex 是一个面向文本、页面和文档的高效向量化索引、检索与生成 (RAG) 系统。它集成了最新的 AI 内容检索技术与工业级 RAG 全流程，支持多源数据的高效处理与语义增强问答。

## 项目核心功能
- **fixedkeyfiles4：实现工业级 RAG 全流程**
  - 文档预处理、特征提取、索引构建、向量数据库融合检索、语义增强生成
  - 支持大规模分布式部署与多源数据接入

## 主要特性
- 高性能页面索引与检索
- 向量化特征提取
- 自动摘要生成
- 智能语义检索与内容生成（RAG）
- 高扩展性与企业级安全设计
- 适应多场景知识辅助

## 安装与运行

```shell
pip install -r requirements.txt
python main.py
```
